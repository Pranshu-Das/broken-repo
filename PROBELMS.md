# Problems Found in the Repository

1. Unnecessary files such as `temp.tmp`, `run.log`, and `program.exe` are generated after execution, cluttering the repository.

2. The definitive purpose wasn't clear in the total output generated.

3. The creation of "program.exe" text file conflicted with the general name given to the produced output exe file after compilation.

4. The repository did not use a proper `.gitignore` file, causing temporary files, logs, cache data, and generated executable files to appear in the working directory.
