# Assignment 1 : Data Management of the Sensor Logger

## Description
This C++ program demonstrates file handling, directory creation, logging, and random data generation.  
It simulates the behaviour of a simple sensor data logging application by generating random sensor readings and organizing them into files and directories.

## Output Directory
The `src/output/` directory is the designated location for all valid program outputs.  
All sensor readings generated during the execution of the program are stored in this directory.

## Valid Output
The valid output file generated by the program is:

- `sensor_data.txt`

All unnecessary temporary files, cache files, logs, and generated executable artifacts have been removed to maintain a clean repository structure.
